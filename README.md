# Clone do Youtube (em Desenvolvimento)


### Arquitetura e Tecnologia:

```
Arquitetura MVVM (MODEL-VIEW-VIEWMODEL);
Retrofit + GSON;
Picasso;
ExoPlayer;
Android custom views;
Android animations
```
